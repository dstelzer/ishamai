# Ishamai: Cuneiform for the web

As seen in [my cuneiform lessons](https://meadstelzer.com/daniel/cuneiform/lessons/cuneiform.html)!

This repository is kind of sparse at the moment, but it was split off from [Cuneiform Tools](https://github.com/dstelzer/cuneiformtools) so it can be used more easily as a submodule. The font files are [generated from that repository](https://github.com/dstelzer/cuneiformtools/blob/master/hantatallas/fonts.py).
